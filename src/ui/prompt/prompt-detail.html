简介

将任何直白、生硬的表达，转化为礼貌、得体且不失分寸的沟通话术。无论是向上级汇报坏消息，还是拒绝不合理的要求，或是提出批评性意见，这个提示词都能帮你找到最合适的表达方式。

#职场沟通 #情商提升 #人际关系 #话术优化

核心功能

话术转换

将直白表达转为委婉得体的沟通方式

场景适配

根据不同场合提供最合适的表达建议

多种风格

提供正式、友好、幽默等多种沟通风格

即时优化

快速生成多个优化版本供选择

使用示例

职场汇报

原始表达：

"项目搞砸了，预算超了50%，时间也延期了。"

优化后：

"关于XX项目，我需要向您汇报一些挑战。目前项目遇到了一些预料之外的困难，导致预算超出原计划50%，交付时间也需要相应调整。我已经准备了详细的问题分析和解决方案..."

拒绝请求

原始表达：

"不行，这个需求太离谱了，做不了。"

优化后：

"感谢您的信任。我仔细评估了这个需求，考虑到目前的资源限制和技术可行性，恐怕短期内难以实现。不过，我有一个替代方案可能会帮助您达到类似的效果..."
